# 🌙 BOOGER CLIENT

BOOGER CLIENT is a next-generation Minecraft client focused on performance, responsiveness, and clean design.

## 🚀 Features (Planned)

* High FPS optimization
* Advanced HUD system
* Modular mod support
* Clean and modern UI
* PvP-focused improvements

## 🧠 Goal

To create a client that competes with Lunar Client while offering better performance and more flexibility.

## 📁 Structure

* `client/` → Core Minecraft client modifications
* `launcher/` → Custom launcher & startup system

## ⚙️ Status

🚧 In early development

## 📌 Notes

This project is actively being developed and will evolve over time.
